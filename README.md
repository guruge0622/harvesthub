# HarvestHub

Simple static e-commerce frontend for local XAMPP/PHP hosting.

## Overview

HarvestHub is a static website project (HTML/CSS/JS) with product pages and a small cart UI. It's intended to be run from a local web server such as XAMPP.

## Files

- `index.html` — main landing page
- Other pages: `bakery.html`, `dairy.html`, `fruits.html`, `vegetables.html`, etc.
- Assets: images and `add to cart/` helper files

## Run locally

1. Ensure XAMPP (Apache) is running.
2. Place the project in your web root (example path for Windows): `C:\xampp\htdocs\harvesthub`.
3. Open in browser: `http://localhost/harvesthub/index.html`.
